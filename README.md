# react-image-hotspots

React component for rendering images with zoom controls and hotspots.

## Prerequisites

- [Node.js 10](https://nodejs.org/dist/latest-v10.x/)

## Test

From project directory, run:

```
npm test
```

## Run

From project directory, run:

```
npm start
```

## Build

From project directory, run:

```
npm run build
```
